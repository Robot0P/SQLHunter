功能特性

专为安全研究人员和渗透测试工程师设计的一站式 SQL 注入检测解决方案

智能扫描引擎

基于 sqlmap 的强大检测能力，提供直观的图形化操作界面。支持多种注入技术，自动识别数据库类型和后端环境。

  • 支持 Boolean、Error、Union、Stacked、Time-based 等多种注入技术
  • 自动识别 MySQL、PostgreSQL、Oracle、MSSQL 等主流数据库
  • 批量目标导入，任务队列自动管理

智能代理池

内置代理池管理系统，支持从多个来源自动获取代理，定期健康检查，自动剔除失效代理。

  • 支持 HTTP/HTTPS/SOCKS5 多种代理协议
  • 自动轮换策略，避免被目标封禁
  • 一键集成 Tor 匿名网络，实时显示链路节点

可视化配置

告别复杂的命令行参数，所有配置通过直观的界面完成。内置多种扫描预设，一键切换扫描策略。

  • 快速扫描、深度扫描、WAF 绕过等预设模式
  • 自定义预设保存，常用配置一键加载
  • 实时命令预览，随时查看生成的 sqlmap 命令

结果分析

扫描结果智能解析，可视化展示发现的漏洞信息。支持 AI 辅助分析，自动生成漏洞评估报告。

  • 数据库结构树可视化展示
  • AI 智能分析扫描结果，提供修复建议
  • 扫描历史记录，随时回顾和对比

界面预览

主控制台

简洁的扫描控制界面，左侧配置面板，中央实时输出，右侧结果展示。一目了然的布局设计。

智能代理池

代理管理、健康检查、Tor 网络集成。实时显示当前 IP、代理状态和 Tor 链路节点信息。
